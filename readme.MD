# Shell Script

**steps:**
* Create repo in GitHub
* Clone repo in local laptop
* Start devopling
* Add the files to temp area
```
git add <file-name>
```
* Commit the changes
```
git commit -m "why-you-commited"
```
* Push the changes to GitHub
```
git push -u origin main
```

**naming convention: use smallcase with Hypons(-)**
